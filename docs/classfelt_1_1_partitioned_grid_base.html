Detailed Description

template<class Derived>
class felt::PartitionedGridBase< Derived >

Base class for spatially partitioned grid storing arbitrary values.

Uses multiple inheritance from child grid (MixinType) class, spoofing the signature, and PartitionBase for actual storage in a (single-level) tree hierarchy, i.e. PartitionedGridBase -> Child -> LeafType.
