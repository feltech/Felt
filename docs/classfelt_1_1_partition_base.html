Detailed Description

template<class Derived>
class felt::PartitionBase< Derived >

Base class for spatially partitioned structures.

A MultiTrackedGrid is used to store and track arbitrary Child structures. The grid as a whole has a size, which is the size of a Child multiplied by the size of the MultiTrackedGrid. The Child might not be another grid type (e.g. see PartitionedArray).
