#define FELT_SURFACE_OMP_MIN_CHUNK_SIZE   32

Minimum number of (active) spatial partitions required before enabling OpenMP loop parallelism.

OpenMP loop parallelisation has signficant overhead (a few milliseconds), so the amount of work done by each thread must be enough to warrant this overhead.
