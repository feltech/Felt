Detailed Description

template<class Derived>
class felt::LookupGridBase< Derived >

Base class for a lookup grid.

Array elements store grid positions and grid nodes store array indices. Each grid node holds an n-tuple of indices, and there are n arrays associated with this lookup grid. An alternative would be to use multiple grids each tracking a single list, but a multi-list structure allows related lists to be spatially tracked in contiguous memory.
