)
inline

Reset and conditionally deactivate children.

All child grids will be reset, but they will not be deactivated and removed from tracking if the given master grid is currently tracking them.

This is an optimisation to ensure spatial partitions that are 'paired' do not get constantly created and destroyed unnecessarily.

Parameters
grid_master_PartitionedGrid-like grid to use as a "master"/"mask".
list_idx_the tracking list id to reset.
