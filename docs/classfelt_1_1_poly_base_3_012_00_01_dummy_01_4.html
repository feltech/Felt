template<class Dummy >
const Vec2i felt::PolyBase< 2, Dummy >::SpxGridPosOffset
staticprotected

Offset to normalise marching squares/cubes corner ordering.

The marching cubes implementation marches in the negative z-axis, but positive x and y axes. Hence an offset is required so that the negative z-axis marching is compensated by shifting the calculation in the +z direction by one grid node. (NOTE: has no effect for 2D).
